# Finance Tracker CLI

A simple command-line finance tracker built with JavaScript.

This project simulates a basic bank account system where users can:

- Deposit money
- Withdraw money
- Track transaction history
- Validate inputs
- Prevent actions on inactive accounts

i built this to reinforce core JavaScript fundamentals like objects, state management, methods, conditionals, and loops.

---

## Features

- Account state management
- Transaction recording
- Input validation
- Action rejection when account is inactive
- Transaction history logging

## Run Locally

Clone repo:

```
git clone https://github.com/ghxstovshad/finance-tracker-cli.git
cd finance-tracker-cli
```

Run:

```
node build.js
```

(or open in browser console)
